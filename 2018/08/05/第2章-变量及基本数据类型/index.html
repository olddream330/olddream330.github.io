2.1 变量

  • lazy 只能修饰val

2.2 基本数据类型

  • Byte
  • Short
  • Int
  • Long
  • Char：单引号
  • String：双引号，三个双引号原样输出
  • Float：加F或f
  • Double：指数表示法，0.314529e1 == 0.314529*10
  • Boolean
  • 首字母大写，所有值类型都是对象

2.3 基本类型操作

  • Scala基于内容比较，Java基于引用比较
  • Scala会将String转换为StringOps，调用reverse、map、drop、slice等自动进行隐式转换
