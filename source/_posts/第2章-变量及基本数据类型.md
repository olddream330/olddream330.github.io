## 2.1 变量
- lazy 只能修饰val


```scala211
// String return null
var s: String = _
// Int Float Double return 0
var i: Int = _
// Char: blank(REPL return ?)
var c: Char = _
```

## 2.2 基本数据类型
- Byte
- Short
- Int
- Long
- Char：单引号
- String：双引号，三个双引号原样输出
- Float：加F或f
- Double：指数表示法，0.314529e1 == 0.314529*10
- Boolean
- 首字母大写，所有值类型都是对象

## 2.3 基本类型操作
- Scala基于内容比较，Java基于引用比较
- Scala会将String转换为StringOps，调用reverse、map、drop、slice等自动进行隐式转换

## 2.4 运算符的优先级

|优先级|符号|
|:---|:---|
|1|*、/、%|
|2|+、-|
|3|::、:::|
|4|<=、>=、==、=、!、!=|
|5|<、>、<<、>>、<<<、>>>|
|6|&、&&|
|7|^|
|8|位或、逻辑或|
